MagicCLOCK & M4ipswtools

magicclock


Main features 主要特点

• Read and write the syscfg of iWatch 读取和写入iWatch的syscfg
• Make NAND replacements easier 让NAND的更换更容易
• Diagnosis of iWatch through MagicCFG Diagnostics 通过MagicCFG诊断法对iWatch进行诊断
• Check voltage, read PMU registers, test hardware 检查电压，读取PMU寄存器，测试硬件

Download and installation 下载和安装

How to install 如何安装
• Download the software 下载软件
Download 下载
• Place into /Applications folder 放到/应用程序文件夹中
• Install libusb over brew.sh 在brew.sh上安装libusb
• Execute the software 执行软件

Free licensing 免费许可

The license of MagicClock is same as of M4iPSWTools. You just need to register email in software again and insert license code. MagicClock的许可证与M4iPSWTools的相同。你只需要在软件中再次注册电子邮件并插入许可证代码。

How to get license 如何获得许可证

• Go to activate.magicawrt.com 转到activate.magicawrt.com
• Enter name, working email address, UUID of mac,  SN of MagicAWRT 
  输入姓名、工作电子邮件地址、Mac的UUID、MagicAWRT的SN
• Wait for license code. Can take up to 24h... 等待许可证代码。可能需要长达24小时...
• Activate the software 激活软件
